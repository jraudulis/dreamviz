# DreamViz

DreamViz is a full-stack web application that generates AI images from text prompts.

## Features
- User authentication & protected routes
- AI image generation from prompts
- Generation history
- Download & share images
- Backend-enforced usage limits
- Error & loading state handling

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: PostgreSQL
- Auth: JWT
- AI: OpenAI API
